# 📖 Overview

The **Indian Bovine Recognition System** is an AI-powered web application that automatically identifies **Indian cattle and buffalo breeds** from uploaded images.

The project combines **Computer Vision**, **Deep Learning**, and a lightweight web interface to deliver fast and accurate breed recognition. It was developed to support livestock identification, breed documentation, smart farming, and digital agriculture initiatives.

The application consists of:

- 🧠 Python Backend
- 🤖 YOLOv8 Detection Model
- ⚡ ONNX Runtime Inference
- 🌐 HTML, CSS & JavaScript Frontend
- 📖 Breed Information Database
